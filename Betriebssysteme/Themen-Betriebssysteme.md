# Themeliste Betriebssysteme

Beim Thema Betriebssysteme kann Sie folgendes in der Klausur erwarten:

1. Unix
    - Grundlegende Aspekte
    - Navigation im Dateisystem
    - Arbeiten mit Dateien
    - NICHT: Dateizugriffrechte 

2. Vim
    - Modi (Normalmodus, Einfügen, Befehlszeile)
    - Wechsel zwischen den Modi
    - Speichern und Beenden

3. Konzepte
    - Speicherhierarchie
    - Schichtenmodell (Hardware, Treiber, Kernel, Systemdienste, Anwendungen)
    - Ressourcenverwaltung (Interrupts, Multitasking, Speicherverwaltung)
    - Anforderungen an Smartphone-Betriebssysteme
    - Backups (Incremental Backup, Offsite Backup, 3-2-1-Regel)
